## üíª Installation and Usage

### System Requirements:
- **Node.js** >= 18.x

### Installation Guide:
1. **Clone the Project:**
   ```bash
   git clone https://github.com/diepdo1810/mixed-hotpot-toolkit.git
   cd mixed-hotpot-toolkit
   ```

2. **Install Required Libraries:**
   ```bash
   npm install
   ```

3. **Run the Application:**
   ```bash
   npm start
   ```
